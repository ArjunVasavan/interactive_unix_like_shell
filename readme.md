## Description

Creating an mini shell which replicates actual unix not perfectly only
half 

## Installation

Clone the repository:
```bash
git clone https://github.com/ArjunVasavan/interactive_unix_like_shell
cd interactive_unix_like_shell
```

## Compilation

```bash
make
make run
make clean
```


## Requirements

- GCC compiler
- Linux/Unix system
